# Overview

This is simple calculator written using stack. It supports brackets and operations '+' '-' '*' '/' 

# Installation
```bash
$ make 
$ sudo make isntall
```

# Usage

```bash
$ stack-calculator "(2 + 2) * 2"
$ (2 + 2) * 2 = 8.00
```
# Uninstall
```bash
$ make clean
$ sudo make uninstall
```
